# PerlDemo

Accepts a nucleotide sequence as input, finds the top hits via blast
and plots GC content of these aligned sequences.

##Modules to be installed :
* HTTP::Server::Simple::CGI
* CGI
* URI::Escape
* LWP::UserAgent
* HTTP::Request::Common
* LWP::Simple
* MIME::Base64
* GD::Graph::bars

##Input :
Nucleotide FASTA Sequence
